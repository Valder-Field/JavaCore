# 2.1 安装Java开发工具包

JDK：Java Development Kit，用于编写Java程序

JRE：Java Runtime Environment（Java运行时环境），用于运行Java程序。它包含虚拟机但不包含编译器，为不需要编译器的用户提供。

Java SE 8u31：Java SE 8的第31次更新

安装源文件和文档：Java的一些核心源码是基于C写的

# 2.2 使用命令行工具

- 编译：`javac xx.java`
- 运行：`java xx`
